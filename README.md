# 🌤️ Cozy Stroll (tên tạm)

Game 3D **cozy, sáng sủa** — chạy vòng vòng khám phá một thị trấn nhỏ ấm áp cho vui. Không áp lực, không thua cuộc.

## Vibe
- Phong cách **stylized / low-poly** màu pastel, ánh sáng ấm (golden hour).
- Nhân vật chạy/đi bộ tự do quanh map nhỏ.
- Âm thanh nhẹ nhàng, lo-fi.

## Công nghệ
| Thành phần | Lựa chọn |
|---|---|
| Engine | **Unity 6 (6000.1.10f1)** |
| Render | URP (Universal Render Pipeline) |
| AI assistant | Claude Code + [CoplayDev/unity-mcp](https://github.com/CoplayDev/unity-mcp) (~9.9k ⭐) |
| Ngôn ngữ script | C# |

1. Đọc [SETUP.md](SETUP.md), cài MCP.
2. Tạo Unity project URP tên `CozyStroll` trong thư mục này.
3. Nói với Claude Code: *"dựng scene thị trấn cozy và cho nhân vật chạy vòng vòng"*.
