# 📦 ASSETS — Danh sách & nguồn free

Ưu tiên asset **low-poly / stylized** màu sáng để giữ vibe cozy và nhẹ máy.

## Cần có (MVP)
| Loại | Mô tả | Nguồn gợi ý |
|---|---|---|
| Nhân vật | Model + animation (idle/walk/run/jump) | [Mixamo](https://www.mixamo.com) (free, auto-rig) |
| Môi trường | Nhà, cây, đường low-poly | [Kenney.nl](https://kenney.nl/assets) (CC0), Unity Asset Store free |
| Skybox | Bầu trời sáng/pastel | [Kenney Skybox], Asset Store "Fantasy Skybox FREE" |
| Âm thanh | Nhạc lo-fi + SFX bước chân | [Freesound](https://freesound.org), [Kenney Audio] |
| Font UI | Font tròn dễ thương | [Google Fonts](https://fonts.google.com) (Nunito, Quicksand) |

## Nguồn CC0 / free uy tín
- **Kenney.nl** — toàn bộ CC0, cực hợp low-poly cozy. ⭐ Khuyên dùng đầu tiên.
- **Mixamo** — nhân vật + animation miễn phí (Adobe).
- **Unity Asset Store** — lọc theo *Free* + *Pricing: Free*.
- **Poly Pizza** (poly.pizza) — model low-poly CC.
- **Freesound / Pixabay** — nhạc & SFX free.

## Quy ước import
- Bỏ asset vào `Assets/Art/`, `Assets/Audio/`, `Assets/Characters/`.
- Material dùng shader **URP/Lit** (vì project là URP).
- Giữ poly thấp để chạy mượt — đây là game cozy, không cần realistic.
